# RetailFlow Dashboard (Angular)

Interactive dashboard consuming the RetailFlow API.

## Views
- **Overview** — supply chain KPIs at a glance
- **Category Analysis** — sales by category and store over time
- **Forecast Explorer** — interactive demand forecast per product

## Setup
```bash
npm install
ng serve
```

Built with Angular 17 + Chart.js. Scaffolding added in Phase 4.
